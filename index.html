Overview

I developed a Cloud Observability Platform designed to monitor real-time logs and metrics of multiple applications running on a Kubernetes cluster. The platform uses FluentD to capture and transmit logs, which are then stored, processed, and displayed on a user-friendly, customizable dashboard built from scratch.

Key Features

Real-Time Log Monitoring

Continuously tracks and displays logs from applications deployed in the Kubernetes cluster, providing timely insights for troubleshooting.

Customizable Dashboard

A visually appealing and interactive dashboard allows you to filter, search, and analyze application and database logs in a streamlined way.

Log Collection with FluentD

Implemented FluentD DaemonSets for seamless log collection and forwarding to the backend service.

Scalable & Extensible

The platform is designed to handle logs from multiple sources, making it adaptable for growing cloud environments.
